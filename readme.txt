If a user is from the EU and GDPR is giving you headaches, just send the visitor away.

NOTE: This plugin makes an external API request to wipmania.com and, as per their terms, identifies the requesting site (your blog) and, of course, your visitors IP.

== Description ==

Redirects site visitors if they're from the EU

== Installation ==

1. In your WordPress admin area, go to Plugins then choose Add New
2. Search for this plugin, Thanks, GDPR
3. Click the Install button (or, upload the zip file via Upload Plugin)
4. Click the Activate button

== Frequently Asked Questions ==

= What kind of redirect do you use?

302 - link juice remains on your site.

= How do I control which countries are redirected?

You don't - unless you edit the plugin PHP file directly.
